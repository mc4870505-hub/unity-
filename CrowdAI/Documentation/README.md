# Crowd AI - Asset для Unity

## Описание
**Crowd AI** - это легковесный и производительный ассет для симуляции поведения толпы в Unity, основанный на алгоритме Boids. Позволяет создавать реалистичное поведение групп персонажей с минимальными настройками.

## Возможности
- ✅ **Алгоритм Boids**: Реализация трех основных правил поведения (сплочение, разделение, выравнивание)
- ✅ **Избегание препятствий**: Динамическое обнаружение и обход препятствий
- ✅ **Масштабируемость**: Поддержка от 10 до 500+ агентов
- ✅ **Гибкая настройка**: Полная kontrolля над параметрами поведения через Inspector
- ✅ **Визуальная отладка**: Gizmos для радиусов восприятия и областей спавна
- ✅ **Динамическое управление**: Добавление/удаление агентов во время выполнения

## Структура проекта
```
CrowdAI/
├── Scripts/
│   ├── CrowdBoid.cs              # Основной компонент агента
│   ├── CrowdManager.cs           # Менеджер создания и управления толпой
│   └── CrowdObstacleAvoidance.cs # Компонент избегания препятствий
├── Prefabs/                      # Префабы агентов (создаются пользователем)
├── Materials/                    # Материалы для визуализации
└── Documentation/                # Документация
```

## Установка
1. Скопируйте папку `CrowdAI` в директорию `Assets` вашего проекта Unity
2. Убедитесь, что у вас установлена Unity 2019.4 или новее
3. Никаких дополнительных зависимостей не требуется

## Быстрый старт

### Шаг 1: Создание агента
1. Создайте пустой GameObject или используйте 3D модель персонажа
2. Добавьте компонент `CrowdBoid`
3. Настройте параметры (скорость, радиус восприятия и т.д.)
4. Сохраните как префаб в папку `Prefabs`

### Шаг 2: Настройка менеджера толпы
1. Создайте пустой GameObject в сцене
2. Добавьте компонент `CrowdManager`
3. Перетащите префаб агента в поле `Agent Prefab`
4. Настройте количество агентов и область спавна
5. (Опционально) Создайте целевую точку и назначьте её в `Global Target`

### Шаг 3: Запуск
Нажмите Play и наблюдайте за поведением толпы!

## Параметры компонентов

### CrowdBoid
| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| Max Speed | Максимальная скорость агента | 5.0 |
| Acceleration | Ускорение агента | 2.0 |
| Neighbor Radius | Радиус восприятия соседей | 3.0 |
| Cohesion Weight | Сила стремления держаться вместе | 1.0 |
| Separation Weight | Сила избегания столкновений | 2.0 |
| Alignment Weight | Сила выравнивания направления | 1.5 |
| Min Separation Distance | Минимальное расстояние между агентами | 1.0 |
| Target | Целевая точка движения | null |
| Target Weight | Сила притяжения к цели | 2.0 |

### CrowdManager
| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| Agent Prefab | Префаб агента толпы | null |
| Agent Count | Количество агентов (10-500) | 50 |
| Spawn Area Size | Размер области спавна | 20.0 |
| Global Target | Общая цель для толпы | null |
| Show Gizmos | Показывать отладочные элементы | true |

### CrowdObstacleAvoidance
| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| Obstacle Detection Radius | Дистанция обнаружения препятствий | 2.0 |
| Avoidance Weight | Сила избегания препятствий | 3.0 |
| Obstacle Layers | Маска слоев для препятствий | All |
| Ignore Agents | Игнорировать других агентов | true |

## Примеры использования

### Динамическое изменение количества агентов
```csharp
CrowdManager manager = FindObjectOfType<CrowdManager>();
manager.SetAgentCount(100); // Изменить на 100 агентов
```

### Добавление агента в конкретную позицию
```csharp
CrowdManager manager = FindObjectOfType<CrowdManager>();
manager.AddAgent(new Vector3(10, 0, 5));
```

### Интеграция с избеганием препятствий
1. Добавьте компонент `CrowdObstacleAvoidance` к префабу агента
2. Настройте слои препятствий в Unity (Edit → Project Settings → Tags and Layers)
3. Назначьте нужный слой в параметре `Obstacle Layers`

## Оптимизация
Для работы с большим количеством агентов (200+):
- Используйте простые коллайдеры (SphereCollider вместо MeshCollider)
- Включите GPU Instancing для материалов
- Рассмотрите использование Object Pooling для частого создания/удаления
- Настройте LOD (Level of Detail) для моделей персонажей
